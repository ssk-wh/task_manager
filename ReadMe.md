一个简单的任务管控终端，允许从远程接收任务执行并反馈任务执行结果



任务类型

| 类型       | 取值 | 说明                       |
| ---------- | ---- | -------------------------- |
| Bash       | 1    | bash命令                   |
| Lua        | 2    | lua命令                    |
| Python     | 3    | python命令                 |
| NoCallBack | 4    | 直接执行，不需回复执行状态 |
| Crontab    | 5    | 定制任务                   |



任务字段

| 字段         | 说明                                     |
| ------------ | ---------------------------------------- |
| task_type    | 任务类型，决定了如何执行task_command命令 |
| task_id      | 唯一的身份ID，用于标识身份               |
| task_command | 任务的具体命令                           |



程序模块

| 模块           | 说明             |
| -------------- | ---------------- |
| TaskCache      | 负责任务状态缓存 |
| TaskHandler    | 执行任务         |
| TaskDispatcher | 任务的分发       |
| TaskManager    | 整体的任务管理   |
